## Установка 

1. Клонирование репозитория 

```git clone https://github.com/Unicorn-Naughty/WeatherApp.git```

2. Переход в директорию WeatherApp

```cd WeatherApp```

3. Установка зависимостей

```npm install ```

6. Запуск для демонстрации возможностей WeatherApp

```npm run dev```
